## 개선할 점

### 1. CSS와 JS의 보드 크기 이중 정의

- JS: `COLS = 10`, `ROWS = 20`
- CSS: `repeat(10, 24px)`, `repeat(20, 24px)`

지금은 일치하지만, 한쪽만 수정하면 불일치가 생길 수 있습니다. CSS 변수(`--cols`, `--rows`) 도입은 이후 단계에서 고려할 만합니다.

### 2. 시작 / 재시작 버튼 동작 동일

- 두 버튼 모두 `resetGame()`만 호출
- 프롬프트 0에서는 문제 없으나, 이후 게임 진행이 시작되면 역할 구분이 필요해짐

### 3. 점수가 데이터 레이어에 없음

- `updateScore(0)`으로 DOM만 갱신, `score` 변수 없음
- 점수 로직 추가 시 `let score = 0` 패턴으로 분리 권장

### 4. `renderBoard()`가 데이터·DOM을 한 함수에서 처리

- 골격 단계에서는 단순해 괜찮음
- 프롬프트 1 이후 `drawPiece()` 분리 시 개선 예정

### 5. README 파일 구조에 `docs/`, `.cursor/` 미반영

- 현재 README 트리는 4개 파일만 표시
- 리뷰 문서·명령 파일이 추가되면 README 갱신 필요
